<div id="matter">
	



Cloud computing is a virtualization-based technology that allows us to create, configure, and customize applications via an internet connection. The cloud technology includes a development platform, hard disk, software application, and database.<br>

The term cloud refers to a network or the internet. It is a technology that uses remote servers on the internet to store, manage, and access data online rather than local drives. The data can be anything such as files, images, documents, audio, video, and more.<br>

There are the following operations that we can do using cloud computing:<br>

Developing new applications and services<br>
Storage, back up, and recovery of data<br>
Hosting blogs and websites<br>
Delivery of software on demand<br>
Analysis of data<br>
Streaming videos and audios<br>

Small as well as large IT companies, follow the traditional methods to provide the IT infrastructure. That means for any IT company, we need a Server Room that is the basic need of IT companies.<br>

In that server room, there should be a database server, mail server, networking, firewalls, routers, modem, switches, QPS (Query Per Second means how much queries or load will be handled by the server), configurable system, high net speed, and the maintenance engineers.<br>

To establish such IT infrastructure, we need to spend lots of money. To overcome all these problems and to reduce the IT infrastructure cost, Cloud Computing comes into existence.<br>

Cloud Computing
Characteristics of Cloud Computing
The characteristics of cloud computing are given below:<br>

1) Agility:<br>

The cloud works in a distributed computing environment. It shares resources among users and works very fast.<br>

2) High availability and reliability:<br>

The availability of servers is high and more reliable because the chances of infrastructure failure are minimum.<br>

3) High Scalability:<br>

Cloud offers "on-demand" provisioning of resources on a large scale, without having engineers for peak loads.<br>

4) Multi-Sharing:<br>

With the help of cloud computing, multiple users and applications can work more efficiently with cost reductions by sharing common infrastructure.<br>

5) Device and Location Independence:<br>

Cloud computing enables the users to access systems using a web browser regardless of their location or what device they use e.g. PC, mobile phone, etc. As infrastructure is off-site (typically provided by a third-party) and accessed via the Internet, users can connect from anywhere.<br>

6) Maintenance:<br>

Maintenance of cloud computing applications is easier, since they do not need to be installed on each user's computer and can be accessed from different places. So, it reduces the cost also.<br>

7) Low Cost:<br>

By using cloud computing, the cost will be reduced because to take the services of cloud computing, IT company need not to set its own infrastructure and pay-as-per usage of resources.<br>

8) Services in the pay-per-use mode:<br>

Application Programming Interfaces (APIs) are provided to the users so that they can access services on the cloud by using these APIs and pay the charges as per the usage of services.<br>
Before learning cloud computing, you must have the basic knowledge of computer fundamentals.
